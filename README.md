# RayTracingIn1Week

Use CMAKE to build the application.

## Coding on Windows
### Before writing code
Since this project is designed to be cross platform, set the `autocrlf` in git config
```powershell
# Set locally
git config core.autocrlf true

# Set globally
git config core.autocrlf true
```

### Running the project

After building run the application in this way:

```powershell
.\main.exe > image.ppm
```
